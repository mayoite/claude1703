# Root Legacy Archive

This folder contains non-runtime root files moved out of the repository root during cleanup.

## Buckets
- `css/`: legacy standalone CSS files
- `js/`: legacy standalone JS files
- `html/`: one-off HTML snapshots/scrapes
- `data/`: generated JSON audit artifacts
- `tools/`: one-off migration/verification scripts
- `assets/`: zipped design/font dump files

## Notes
- Runtime app code remains in `app/`, `components/`, `lib/`, `public/`.
- Global stylesheet source of truth remains `app/globals.css`.
