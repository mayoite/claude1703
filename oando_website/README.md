# Oando Website Delivery Pack

Use `00_MASTER_INDEX.md` as the primary entry point.

## Folder Guide
- `01_sql/migrations` - Supabase migration SQL files
- `01_sql/plans` - generated SQL plans (alias/reconcile)
- `02_reports/audits` - audit outputs
- `02_reports/reconcile` - dry-run reconcile outputs
- `02_reports/plans` - slug policy and planning reports
- `02_reports/walkthroughs` - operational runbooks
- `03_charts` - PNG/SVG flowcharts
- `04_code` - copied code changes for reference
- `05_config` - copied config files
- `06_scripts` - helper scripts
- `99_bundles` - final bundle + archive
